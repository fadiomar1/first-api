# AI Study Assistant

مشروع ويب بسيط يدخل فيه الذكاء الاصطناعي: المستخدم يكتب موضوعا أو سؤالا، والمساعد يرجع شرحا منظما مع مثال وأسئلة مراجعة.

## التشغيل

1. افتح هذا المجلد في الطرفية.
2. ضع مفتاح OpenAI API:

```powershell
$env:OPENAI_API_KEY="your_api_key_here"
```

3. شغل المشروع:

```powershell
npm start
```

4. افتح:

```text
http://localhost:3000
```

## تغيير النموذج

النموذج الافتراضي موجود في `server.js` ويمكن تغييره أيضا من متغير البيئة:

```powershell
$env:OPENAI_MODEL="gpt-5.5"
```

إذا لم يكن النموذج متاحا في حسابك، غيّر `OPENAI_MODEL` إلى نموذج متاح لك.

## كيف تطوره لاحقا

- إضافة رفع ملفات PDF حتى يشرح منها.
- حفظ الأسئلة والإجابات في قاعدة بيانات.
- إضافة تسجيل دخول للمستخدمين.
- إضافة وضع "اختبرني" لتوليد أسئلة فقط.
- إضافة دعم الصوت: المستخدم يتكلم والمساعد يرد.
